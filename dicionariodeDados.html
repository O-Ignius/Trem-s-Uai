Dicionário de Dados - E-commerce

Tabelas

Tabela: `endereco`
Campo Tipo Descrição
id INT Identificador único do endereço
cep INT Código de Endereçamento Postal (CEP)
rua VARCHAR(255) Nome da rua
complemento VARCHAR(255) Informações complementares do endereço
logradouro VARCHAR(100) Tipo de logradouro (Avenida, Rua, etc.)
bairro VARCHAR(100) Nome do bairro
cidade VARCHAR(100) Nome da cidade
estado VARCHAR(100) Nome do estado
numero INT Número do endereço
Tabela: `cliente`
Campo Tipo Descrição
id INT Identificador único do cliente
nome VARCHAR(100) Nome completo do cliente
cpf VARCHAR(14) Cadastro de Pessoa Física (CPF) do cliente
email VARCHAR(100) Endereço de e-mail do cliente
senha VARCHAR(100) Senha de acesso do cliente
telefone VARCHAR(15) Número de telefone do cliente
dataNascimento DATE Data de nascimento do cliente
nacionalidade VARCHAR(50) País de origem do cliente
genero VARCHAR(10) Gênero do cliente
endereco_id INT Referência ao endereço do cliente
Tabela: `vendedor`
Campo Tipo Descrição
id INT Identificador único do vendedor
nome VARCHAR(100) Nome completo do vendedor
cpf VARCHAR(14) Cadastro de Pessoa Física (CPF) do vendedor
cnpj VARCHAR(18) Cadastro Nacional de Pessoa Jurídica (CNPJ), caso aplicável
email VARCHAR(100) Endereço de e-mail do vendedor
senha VARCHAR(100) Senha de acesso do vendedor
telefone VARCHAR(15) Número de telefone do vendedor
dataNascimento DATE Data de nascimento do vendedor
nacionalidade VARCHAR(50) País de origem do vendedor
genero VARCHAR(10) Gênero do vendedor
endereco_id INT Referência ao endereço do vendedor
Tabela: `produto`
Campo Tipo Descrição
id INT Identificador único do produto
nome VARCHAR(100) Nome do produto
descricao TEXT Descrição detalhada do produto
valor_atual DOUBLE Preço unitário do produto
estoque INT Quantidade disponível em estoque
custo_atual DOUBLE Custo atual do produto
vendedor_id INT Referência ao vendedor do produto
Tabela: `avaliacao`
Campo Tipo Descrição
id INT Identificador único da avaliação
nota INT Nota atribuída ao produto (1 a 5)
comentario TEXT Comentário do cliente sobre o produto
data DATE Data em que a avaliação foi realizada
cliente_id INT Referência ao cliente que fez a avaliação
produto_id INT Referência ao produto avaliado
Tabela: `carrinho`
Campo Tipo Descrição
id INT Identificador único do carrinho
cliente_id INT Referência ao cliente que criou o carrinho
data_criacao DATETIME Data e hora de criação do carrinho
Tabela: `itenscarrinho`
Campo Tipo Descrição
carrinho_id INT Referência ao carrinho
produto_id INT Referência ao produto
quantidade INT Quantidade do produto no carrinho
preco_unitario DOUBLE Preço do produto no momento da adição
Tabela: `venda`
Campo Tipo Descrição
id INT Identificador único da venda
data_hora DATETIME Data e hora em que o pedido foi realizado
total_pago DOUBLE Valor total dos itens no carrinho
tipoPagamento VARCHAR(45) Código do tipo de pagamento escolhido (ex: cartão, boleto, etc.)
pagamento_efetuado TINYINT Indica se o pagamento foi efetivado (1 para pago, 0 para não pago)
cliente_id INT Identificador do cliente que fez a compra (chave estrangeira para `cliente`)
Tabela: `itensvenda`
Campo Tipo Descrição
produto_id INT Referência ao produto vendido (chave estrangeira para `produto`)
venda_id INT Referência à venda (chave estrangeira para `venda`)
quantidade INT Quantidade do produto na venda
preco_praticado DOUBLE Preço do produto no momento da venda
